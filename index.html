CPU SCHEDULER

The CPU SCHEDULER executes various CPU scheduling algorithms, providing users with insights into process scheduling and performance metrics.

Here are step-by-step instructions for using the CPU scheduler simulation:

  • Input Process Details:

  • Enter Process ID, Arrival Time, and Burst Time for each process in the form provided. Select the scheduling algorithm (FCFS, SJF, SRTF, or RR) and specify the Time Quantum for Round Robin if applicable.
  • Click "ADD PROCESS" after filling each process's details to add them to the simulation queue.
  • Once all processes are added, click "OUTPUT" to run the selected scheduling algorithm.
  • Results including Completion Time, Waiting Time, and Turnaround Time will be displayed dynamically in the table below. Average Turnaround Time, Average Waiting Time, and Throughput will be calculated and shown at the bottom.
  • Adjust parameters or add more processes to explore different scenarios.
  • Use browser refresh to clear data and start over if needed. Enjoy exploring different CPU scheduling strategies with our CPU SCHEDULER!
